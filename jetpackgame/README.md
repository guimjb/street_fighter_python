# Jetpack Game - Jetpack Jeongride

This is my project for Programming Design class. The game is inspired by jetpack joyride and flappy bird. It is a 2D side-scroller built with kivy and python.

All dependencies are listed in `requirements.txt`.  
You can install them using pip:
pip install -r requirements.txt

If using thonny pip can be installed by:
Tools -> Manage Packages, search "Kivy"
Click install and run game

To run the game otherwise
From the project root directory, run:
python app/jetpackgame.py
